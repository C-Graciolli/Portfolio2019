Consulting Digital Platform

Consulting Digital Platform is a suite of web applications developed for Dimension Data. Initially a single application, the platform now supports multiple practices.

The platform is a two-sided application with Admin and Presentation Areas.

Regular additions to the platform demanded consistency. I led the redesign and developed the common look and functionality which then needed to be migrated across the product suite, ensuring a consistent user experience.

I produced an evolving set of design components and visual style guidelines.

In order to build a framework that was both portable and scalable, I refactored the mark-up and stylesheets from the ground up and developed front-end code that was both future proof and easy to maintain.

Client: Fusion Life Style
Roles: User Interface designer | Front-end developer

MyFusion Class Booking System

Fusion Life Style is a sports and leisure management organisation that works in partnership with local authorities, schools and colleges, and other public sector organisations, to provide recreational sport, health and fitness activities to local communities.

There were two elements to this project:

  1. Adapt the existing front-end class timetables to be responsive across all devices.

  2. Design and develop a mobile friendly booking system for fitness classes.

Fully Responsive Class Timetables

The main goal of the class timetables redesign was to ensure a seamless experience across all devices.

I was responsible for producing the visuals for the new interface. The timetable was re-designed to display a full week schedule for both desktops and tablets, reducing to a 1 day scrollable schedule on mobile.

Enhancements such as class status and class description were added, enriching user experience within the actual timetable.

New class booking system: MyFusion

MyFusion, came to life as a one stop for all class booking needs, across multiple devices and screen sizes.

The web application is centralised on a dashboard where user has full access and control over available classes: timetables, upcoming bookings, waiting list, previous books and centre details.

Via MyFusion, the user can book and cancel classes on the go.

The app also enables users to control and edit their account details and account preferences.
